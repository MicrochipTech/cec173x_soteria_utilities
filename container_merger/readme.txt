 Conatiner merger Folder - Readme.txt
===============================================================================

This tool is to merge the create container rpmc binary into the soteria GEN3 HEX/BIN file 


************* CEC173x GEN3 and RPMC Flash container Merger Ver: 1.00 ******


Generated the final soteria GEN3 Hex/bin file = final_soteria_hex/soteria_gen3_create_container.bin

----------------------------------------------
1. Copy the Soteria GEN3 hex file
2. Copy the create container hex file from the rpmc tool RPMC_flash_container\rpmc\contatenate_create_container_signature.bin into the create_container_hex directory
3. Run the container_merger.exe in the command prompt
4. It wil generate the final_soteria_hex in hex/bin file into the soteria gen3 spi image gen
