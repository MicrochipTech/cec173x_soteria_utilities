 MCHP - SPI Image generator for GLACIER 
===========================
THIS FOLDER CONTAINS FOLLOWING FILES:
ec384.pem :   ec384 private  key of password ec384
ec384_crt.pem :   ec384 cert pem file 
ec384_csr.pem :   ec384 csr pem file 
ec384_pub.pem :   ec384 public pem file 

ec384_plain_priv.pem : plain ec384 private key
ec384_pub.pem : plain ec 384 public key 

ec384_<number>.pem :   ec384 private  key of password ec384_<number>
ec384_<number>_crt.pem :   ec384 cert pem file 
ec384_<number>_.pem :   ec384 csr pem file 
ec384_pub_<number>_.pem :   ec384 pub pem file
